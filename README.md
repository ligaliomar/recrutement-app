T# Recrutement App

Application NextJS 14 pour la gestion des candidatures.

## Technologies
- Next.js 14 (App Router)
- Ant Design
- Redux Toolkit
- next-intl

## Installation

\`\`\`bash
npm install
npm run dev
\`\`\`

## Pages
- `/candidate` → Formulaire candidat
- `/recruiters` → Liste des candidats
- `/recruiters/[id]` → Détail d'un candidat
